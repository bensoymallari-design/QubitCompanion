# AGENTS.md

## Cursor Cloud specific instructions

Qubit/Resolume Companion is a single local-only Next.js (App Router, React, TypeScript, Tailwind) app. There is only one service.

- Dependencies are installed by the startup update script (`npm install`). Node 22 is available.
- Standard commands live in `package.json`: `npm run dev` (dev server on `0.0.0.0:3000`), `npm run build`, `npm run start`, `npm run lint` (note: `lint` runs `tsc --noEmit`, there is no ESLint).
- Run the dev server in a background/tmux session; it binds `0.0.0.0:3000`. Verify via `curl http://localhost:3000/api/files`.
- The app is designed to talk to Resolume Arena's local HTTP API (`127.0.0.1:8080`). Resolume is not available in this environment, so all `/api/resolume/*` and `/api/system/*` (Windows-only power/close) routes will report disconnected/fail. This is expected; the media upload/browse/settings features work fully without Resolume.
- Local state is written under `storage/` at runtime: `storage/uploads/`, `storage/thumbnails/`, `storage/database.json`, `storage/settings.json`. These are gitignored user data — do not commit them.
- `next-env.d.ts` is auto-regenerated by Next.js (dev mode rewrites its import path to `.next/dev/types`). Do not commit these incidental changes; `git checkout next-env.d.ts` to revert.
